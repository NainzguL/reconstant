# Example

## Create an input file

#### `test.yaml`

```yaml
constants:
  - name: SOME_CONSTANT
    value: 'this is a constant string'
  - name: OTHER_CONSTANT
    value: 42

enums:
  - name: SomeEnum
    values:
      - A
      - B
      - C
  - name: OtherEnum
    values:
      - FOO
      - BAR

outputs:
  python:
    path: autogenerated_constants.py
  javascript:
    path: autogenerated_constants.js
  c:
    path: autogenerated_constants.h
  php:
    path: autogenerated_constants.php
  vue:
    path: autogenerated_vue_constants.js
  java:
    path: AutogeneratedConstants.java
  dart:
    path: autogenerated_constants.dart
```

## Install and run reconstant

```
pip install pyyaml
reconstant test.yaml
```

	A: 0,
	B: 1,

	created() {

};
